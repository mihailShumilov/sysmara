# Prompt

> Copy and paste this into Claude Code, Cursor, Windsurf, or any AI coding agent.

```
I want to build a blog platform using SysMARA. Read BOOTSTRAP.md and follow it.

Here's what I need:

Authors can write blog posts and submit them for review. Editors review
and approve or reject posts before they go live. Posts can be tagged for
categorization. Readers can leave comments, but comments should be
moderated — nothing shows up until a moderator approves it.

Three roles: writer (creates posts), editor (reviews and approves),
admin (can do everything).

All moderation actions should be logged so we have an audit trail.

Set up the full project — specs, build, implementation, tests, and
a running server.
```
