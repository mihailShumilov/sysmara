```
I want to build a SaaS task manager using SysMARA. Read BOOTSTRAP.md
and follow it.

Here's what I need:

Users sign up and create workspaces. Each workspace has members with
roles — owner, admin, and regular member. Tasks belong to a workspace

Members can comment on tasks. Workspace owners can invite and remove
members. When a new workspace is created, the creator becomes the
owner and can invite initial members right away. If something goes
wrong during setup, everything should roll back cleanly.

Set up the full project — specs, build, implementation, tests, and
a running server.
```
